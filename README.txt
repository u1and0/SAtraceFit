# SAtraceFit使用方法
SAtraceによって収集された5分間隔のファイルを集計するpythonスクリプト群の説明を行う。


# データ定義

## データ形式
1000行4列のtxtファイル
`ファイル名=タイムスタンプ(測定時刻yyyymmdd_HHMMSS.txt)`
日付が最初の8桁、アンダースコアを挟んで、時間を6桁で表している。

1列目 : 行数
2列目 : maxホールド
3列目 : 平均値
4列目 : minimumホールド

スタート周波数が行数をxとしたとき、(x*4+22000)/1000で周波数となる。
ここでは2列目


## データの場所
ファイル名confidential.pyに変数rootPathとして格納した。








# 各モジュールの説明


## main.py
<一言紹介>
各モジュールを動かすメインファイル
<簡易動作説明>
引数:なし(confidential.pyから変数を持ってくる)
戻り値:なし(CSVファイルに書き込む)
<使用方法>
コマンドライン上にて
`python main.py <最初の日付> <最後の日付>`
と打ち込む。
日付のフォーマットはyymmdd形式(例えば2015年11月1日=151101と打ちこむ)。



<詳細動作説明>






## datelist.py
<一言紹介>
main.pyから2つの値が渡される
2つの間の日付のリストを返す
<動作説明>
yymmdd形式の6桁をdatetime関数で日付にする
ddate1,2に代入する。
ddate1がddate2になるまで(while)
ddate1を1日ずつ足して、リストに追加する(append)。
<使用方法>
引数 : 最初の日付yymmdd形式、最後の日付yymmdd形式
戻り値 : 最初の日付から最後の日付を入れたリストyymmdd形式
